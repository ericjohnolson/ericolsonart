# ericolsonart

Personal art portfolio site for Eric Olson, built with [Astro](https://astro.build).

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)

## Setup

```bash
npm install
```

## Development

Start the local dev server:

```bash
npm run dev
```

The site will be available at **http://localhost:4321**.

## Production Build

```bash
npm run build
npm run preview
```

`build` outputs static files to `dist/`. `preview` serves the built site locally for verification.

## Type Checking

```bash
npx astro check
```
